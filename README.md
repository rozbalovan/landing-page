# 🌐 Landing Page

> Personal landing / link-in-bio page for **Rozbalovan**.

A clean, single-page HTML site with social links, project showcases, and contact info. Built as a lightweight link-in-bio alternative to services like Linktree.

## ✨ Features

- **Minimal, responsive design** — works on mobile & desktop
- **Social links** — X/Twitter, Reddit, Telegram, Instagram, GitHub
- **Project showcase** — Gold Signal Bot, MexArb Bot, E-Commerce, AI pipeline
- **Contact section** — email link
- **No dependencies** — pure HTML + CSS, zero JavaScript bloat

## 🚀 Live

**[dmytro-belilovskiy.surge.sh](https://dmytro-belilovskiy.surge.sh)**

## 📁 Structure

```
landing-page/
└── index.html   # Single HTML page with embedded styles
```

## 🛠️ Usage

Just open `index.html` in any browser, or deploy to any static hosting:

```bash
# Deploy to Surge.sh
surge ./ landing-page.surge.sh

# Or just open locally
open index.html
```
